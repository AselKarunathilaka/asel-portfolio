DevOps Learning

How This Project Uses DevOps Concepts

Version Control

Git and GitHub store the project code so changes can be tracked, rolled back, and deployed more safely.

Containerization

Docker packages the site in a consistent runtime environment, making deployment cleaner and easier to repeat.

Web Serving

Nginx serves the static website files to the browser from inside the container.

Linux Hosting

The application runs on an Ubuntu server, helping me practice real server management skills.

Deployment Workflow

The project can be updated by pushing changes to GitHub, pulling them on the server, and rebuilding the Docker container.

Future CI/CD

This portfolio creates the foundation for later automation using GitHub Actions and basic continuous integration pipelines.
