# Asel Portfolio Web App

A modern personal portfolio web app built with HTML, CSS, and JavaScript, then deployed using Docker and Nginx on an Ubuntu Linux server.

## Purpose
This project was created to improve practical knowledge in:
- Web development
- Linux server hosting
- Docker
- Nginx
- Git and GitHub
- Beginner DevOps workflows

## Tech Stack
- HTML
- CSS
- JavaScript
- Nginx
- Docker
- Ubuntu Linux
- Git / GitHub
