# Asel Karunathilaka Portfolio

A modern personal portfolio website built with HTML, CSS, and JavaScript, then served with Nginx and Docker.

## Main Goal
This portfolio is designed to present Asel Karunathilaka as a final-year IT undergraduate with Cloud and DevOps as the main focus, while also showcasing software development, machine learning, databases, and security-related learning.

## Features
- Modern responsive UI
- Light and dark theme toggle
- Featured GitHub project showcase
- Personal hero section with profile image
- Contact section with form support
- Dockerized deployment using Nginx
- Smooth animations and section-based layout

## Tech Stack
- HTML
- CSS
- JavaScript
- Docker
- Nginx
- Git / GitHub
